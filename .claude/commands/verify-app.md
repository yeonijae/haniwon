# verify-app

앱이 정상적으로 작동하는지 검증합니다.

## 검증 항목

1. **TypeScript 타입 체크** - `npm run typecheck`
2. **빌드 테스트** - `npm run build`
3. **의존성 확인** - 누락된 패키지가 없는지 확인

## 실행 단계

1. `npm run typecheck`를 실행하여 타입 오류 확인
2. 타입 오류가 있으면 목록과 함께 보고
3. `npm run build`를 실행하여 빌드 성공 여부 확인
4. 빌드 결과 (번들 크기, 경고 등) 보고
5. 전체 검증 결과 요약 제공

## 인자

- `$ARGUMENTS`: 검증 옵션 (선택사항)
  - `quick`: 타입 체크만 실행
  - `full`: 타입 체크 + 빌드 + lint (기본값)

## 출력 형식

검증 결과를 다음 형식으로 보고하세요:

```
## 검증 결과

| 항목 | 상태 | 세부사항 |
|------|------|----------|
| TypeScript | ✅/❌ | 오류 N개 |
| Build | ✅/❌ | 빌드 시간, 번들 크기 |

### 발견된 문제
- (문제가 있으면 목록으로 표시)

### 권장 조치
- (필요한 경우 개선 제안)
```

## 지침

- 오류가 발견되면 구체적인 파일과 라인 번호를 포함하세요
- 빌드 경고도 함께 보고하세요
- 심각한 문제와 경미한 문제를 구분하세요
